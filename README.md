# AIP2
Final project for module: Programming for Artificial Intelligence

## Function
Generates 2 plots of data:
- Line Graph with forecasted and actual power generated by wind given by the contents of the wind-gen.csv.
- Heatmap with the island-wide electricity demand for the period covered by the contents of the system-demand.csv file
